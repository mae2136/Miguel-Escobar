Latest Project - Fully Booked

Fully Booked is a site to see virtual community libraries in the Chicago area. Users can create libraries and mark them with an address and zip code, add books to the library, and checkk books in or out of other libraries in their area. Non-signed up users are able to browse libraries and see what books they contain.

This site was made by the Check Yo Shelves team using Express, Sequelize (MySQL), and Node.js for the Backend, and Handlebars with Bootstrap for the Frontend of the site.
